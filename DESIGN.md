# Colors
Main color palette:
1. Base color #BD6956
2. White #FFFFFF
3. Black #262221
Use mostly white, black, and colors between them (shades). Rarely use the base color for emphasis.

# Font & Text
- Hanken Grotesk from Google Fonts.
- Body text is 1rem. Heading sizes step from that using the multiplier below.
- For paragraphs, line height must be 1.5.
- Character spacing will be -2% globally.
- Every text must be lowercase. Everything.
- Let font size multipler be 1.3x (headings use powers of this factor from 1rem).

# Layout & System
- Max width of pages will be 45rem. They will be centered on the screen horizontally.
- Write everything in rem, not px.
- No background color nor border for pages.
